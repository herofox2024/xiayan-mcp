# xiayan-mcp Web应用测试计划

## 项目分析
xiayan-mcp是一个微信公众号文章发布工具，采用前后端分离架构：
- **Web后端**：FastAPI构建的RESTful API服务，运行在端口8000
- **前端应用**：Vue 3 + Element Plus构建，已部署到后端静态目录
- **核心功能**：凭证管理、文章发布、主题管理、媒体管理

## 测试步骤

### 1. 基础访问测试
- 访问前端应用：http://localhost:8000
- 检查前端页面是否正常加载
- 查看API文档：http://localhost:8000/docs
- 测试API根路径：http://localhost:8000/api

### 2. 凭证管理功能测试
- 进入凭证管理页面
- 测试查看当前凭证信息
- 测试配置微信公众号AppID和AppSecret
- 测试保存到环境变量功能

### 3. 主题管理功能测试
- 进入主题管理页面
- 查看内置主题列表
- 测试主题预览功能
- 测试添加自定义主题
- 测试编辑现有主题

### 4. 媒体管理功能测试
- 进入媒体管理页面
- 测试上传图片素材
- 测试查看媒体列表
- 测试获取永久素材
- 测试删除素材

### 5. 文章发布功能测试
- 进入文章发布页面
- 测试Markdown编辑功能
- 测试主题应用
- 测试预览文章效果
- 测试发布到微信公众号

### 6. 完整流程测试
- 从配置凭证开始
- 上传封面图片
- 编写Markdown文章
- 选择主题
- 预览效果
- 发布到微信公众号

## 预期结果
- 所有页面正常加载
- 所有API端点正常响应
- 所有功能模块正常工作
- 能够成功发布文章到微信公众号

## 测试工具
- 浏览器：Chrome/Edge
- 开发者工具：用于查看网络请求和控制台日志
- API测试工具：可使用Swagger UI（内置在http://localhost:8000/docs）

## 注意事项
- 确保微信公众号AppID和AppSecret正确
- 确保网络连接正常
- 关注控制台日志，及时发现错误信息
- 测试过程中注意保存测试数据，便于问题定位
